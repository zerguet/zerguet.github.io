Instructions for the initial setup

  1. 1. INSTALL JAVA 64bit

    2. If you don't have it already, it can be downloaded from here
    (to avoid issues, please uninstall other java versions)

  2. 2. DOWNLOAD AND INSTALL TLAUNCHER

    3. Download it from here
    Open downloaded file and keep clicking next to finish installation.
    Note: you can use any other launcher, including the official one!

  3. 3. CONFIGURE TLAUNCHER

    4. As shown here :
    • · Open Tlauncher

    • · Click the gear icon on bottom right side of the window and select settings

    • · Change memory allocation to at least 2000MB (more is better)

    • · Click Save

    • · Enter your desired name in bottom left field User Name

    • · Select ForgeOptiFine 1.16.4. from the dropdown list in the middle


  4. 4. ADD MODS

    5. In Tlauncher:
    • · Click Folder icon on the right next to the gear icon

    • · Open mods folder

    • · Download mods from here

    • · Unzip and place downloaded mods to mods folder

    • · Click ENTER THE GAME


  5. 5. ADD THE SERVER AND PLAY

    6. From the main menu:
    • · Click Multiplayer

    • · Click Add Server

    • · For Server Name enter whatever you want

    • · For Server Address enter 147.135.44.229

    • · Click done



    Server should show up on the screen after refresh.
    To play just select the server and click Join Server or just double click it!

    Note: If playing feels claustrophobic or makes you motion sick, increase FOV slider in Minecraft settings!
